# Обновление стартовой страницы анонимного пользователя

## Задача
Обновить лендинг для анонимных пользователей портала обучения B3.

## Источник изменений
Презентация "Платформа Б3" (слайд 2): `C:\Users\PREDATOR\Downloads\Презентация Платформа Б3.pdf`

## Выполненные изменения

### 1. Описание платформы (из презентации)

**Было:**
- Заголовок: "Платформа B3"
- Подзаголовок: "Low-code платформа для создания корпоративных приложений"
- Описание: краткое описание low-code подхода

**Стало:**
- Заголовок: "Платформа Б3"
- Подзаголовок: "Фреймворк для разработки информационных систем в государственном и бизнес-секторе"
- Описание: полный текст из презентации про фреймворк и инфраструктурные решения
- Особенности (из слайда 2):
  - Быстрый запуск государственных решений
  - Сертифицированные модули (ФСТЭК и ФСБ)
  - Отечественное ПО
  - Интеграция с федеральными ИС и СЭД

### 2. Кнопки в шапке

**Было:**
- "Начать обучение" (primary)
- "Войти"

**Стало:**
- "Регистрация" (primary)
- "Войти"

### 3. Блок "Как освоить платформу?"

**Было:** 4 шага
1. Выберите курс
2. Оставьте заявку
3. Практикуйтесь на стенде
4. Получите сертификат

**Стало:** 5 шагов
1. Выберите курс
2. Оставьте заявку
3. Получите доступ к учебным материалам
4. Практикуйтесь на учебном стенде
5. Получите сертификат

### 4. Страница деталей курса

**Удалено:**
- "Порог сертификации: X% от общей оценки"

**Переименовано:**
- "Лабораторный стенд" -> "Учебный стенд"

### 5. Программа курса (список заданий)

**Удалено из каждого задания:**
- Тег "Обязательно"
- Количество баллов (было: "X баллов")

**Оставлено:**
- Порядковый номер и название
- Тип задания
- Срок выполнения (дней)
- Описание

## Затронутые файлы
- `proto/modules/guest.js` - основной файл экранов гостя

## Проверка
Открыть `proto/index-modular.html` в браузере, роль "Гость (аноним)" и проверить:
1. Главную страницу - новое описание платформы
2. Кнопку "Регистрация" вместо "Начать обучение"
3. 5 шагов в блоке "Как освоить платформу?"
4. Страницу деталей любого курса - без порога сертификации, "Учебный стенд"
5. Программу курса - без обязательности и баллов
